Resources

Every great developer has a solid foundation of programming.

Below are some resources (free and paid) online resources that I've found useful in going from novice to experienced web developer

HTML and CSS

HTML and CSS are the foundation of front end development. Take a look at my highly recommended resources.

Dash General Assembly (Free)

General Assembly is one of the biggest technical vocation schools who've been able to build a platform to help people understand how HTML and CSS works. I think this learning platform does a fantastic job of making you retain the information by practicing. It's absolutely free and a great resource if you're looking to start learning HTML & CSS.

Explore

Intro To Javascript & jQuery

Building on your knowledge of HTML and CSS, we're going to work together to bring your website to life by being able to use the power of javascript and jQuery. Make your site interactive and make it fun as well as functional!

Intro To Javascript (Free)

I personally graduated from the Flatiron School and 100% believe in their stellar curriculum. As an intro course to Javascript, you will understand the fundamentals of modern JavaScript web development. It's a free course and I highly recommend it.

Explore

Ruby

Ruby is a great Object Oriented Programming Language which is sleak, clean and easier to pick up. It's also a popular option because it is used to build powerful web applications using Ruby on Rails.

Intro to Ruby (Free)

Intro to Ruby is another free course offered by the Flatiron School. It goes over the fundamentals of ruby to the point where you can build your own game from scratch. When I was going through the curriculum, I build tic-tac-toe that you can play with 2 humans, one human and a computer and two computers.
