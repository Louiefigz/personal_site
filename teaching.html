Teaching Services

I'm currently a Lead Instructor, teaching Web Development basics for a non-profit organization, The Knowledge House. The Knowledge House is geared toward making tech education available to low income students ranging from 9th grade to adults.

I also have experience working on full scale applications for clients; most notable is Momofuku Milk Bar in New York City. While working on that project, I was not only the Lead Developer but the Project Manager and UI Designer. These are a few of the Services that I offer to get you on track to becoming a Full Stack Developer.

Intro To HTML and CSS

If you're looking to master the foundation of Web Development, this is a great crash course to understanding how to create great looking and clean websites with design in mind. Knowing basic HTML and CSS is a neccessary skill to know when thinking about building more complex websites in the future.

Intro To Javascript & jQuery

Building on your knowledge of HTML and CSS, we're going to work together to bring your website to life by being able to use the power of javascript and jQuery. Make your site interactive and make it fun as well as functional!

Git

Manage your projects by yourself or a team of developers using the extremely popular open source version control system on the web. Most professional developers use git everyday and a must have skill as a developer in today's market.

Ruby and Ruby On Rails

Build working web applications using a powerful framework built on Ruby.

I'll teach you how to build anything from a simple Todo list to fully functioning working application.
